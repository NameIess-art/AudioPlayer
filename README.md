# Nameless Audio

Nameless Audio 是一个面向 Android 本地媒体库、多会话 ASMR 播放、长音频、字幕和睡眠计时场景的 Flutter 播放器。界面和业务状态由 Flutter 承载，后台播放、通知控制、息屏保活、定时暂停/自动恢复、媒体扫描与系统权限跳转由 Android 原生能力兜底。

当前版本：`0.8.71+871`<br>
最新发布页：[v0.8.71](https://github.com/NameIess-art/nameless-audio/releases/tag/v0.8.71)<br>
许可证：[MIT](https://github.com/NameIess-art/nameless-audio/blob/main/LICENSE)<br>
隐私说明：[PRIVACY.md](https://github.com/NameIess-art/nameless-audio/blob/main/PRIVACY.md)

> 注意：自 `v0.8.0` 起，Android `applicationId` 已从 `com.example.music_player` 变更为 `com.nameless.audio`。旧包名版本无法直接覆盖安装到新版本上，升级前请先备份并重新安装。

## 主要功能

- 多会话播放：可同时创建多个播放会话；每个会话独立控制播放/暂停、进度、音量、循环模式和左右声道互换。
- 原生后台播放队列：非单曲循环模式会把当前文件夹或跨文件夹队列下沉到 Android Media3/ExoPlayer，息屏和后台状态下也能稳定续播、循环和随机播放。
- Android 后台保活：播放通知、前台服务、WakeLock、精确闹钟、原生播放服务和保活服务协同工作，提升几十小时级别的后台与息屏播放稳定性。
- 层级媒体库导航：支持文件夹分层浏览，自动根据文件树结构生成导航节点，方便管理包含数千个文件的复杂媒体库。
- 精准音频焦点管理：自动处理 Android 系统音频焦点，支持来电暂停、其他媒体抢占后自动暂停及焦点恢复。
- 媒体导入与播放：支持音频和常见视频媒体文件；音频格式包括 `flac`、`wav`、`mp3`、`m4a`、`aac`、`ogg`、`opus`、`3gp`，视频格式包括 `mp4`、`mkv`、`webm`、`mov`、`m4v`、`avi`。
- 三种导入方式：支持导入单个文件、导入单个文件夹、导入整个资料库（父目录下的多一级子目录）。
- 原生优先导入兼容：Android 上优先使用系统文档选择器（SAF）导入 `content://` 资源并持久化读取授权；不可用时保留旧式文件选择回退。
- 音频库管理：支持扫描、刷新、去重、搜索、根目录拖拽排序、左滑移除，以及资料库排除/恢复编辑；新增的根目录会显示在列表最上方。
- 曲库结构持久化：每个曲库的文件夹、子结构和音频条目会单独写入 SQLite，记录正在使用/已排除状态；后续文件夹新增内容可通过刷新同步进曲库。
- 编辑曲库：完整展示曲库结构，排除根文件夹后仍保留下级结构可见；父文件夹被排除时子项同步显示已排除并禁止单独切换，恢复父文件夹后子结构同步恢复使用。
- 分类浏览：可按全部、标签、声优、社团浏览音频；分类数据来自本地作品详情和 DLsite 元数据，分类元素支持长按复制。
- 作品详情与 DLsite：支持编辑作品标题、RJ 号、社团、声优、标签等信息，可按 RJ 号或标题从 DLsite 获取元数据；确认页会在作品标题下显示 RJ 号，并可保存封面到文件夹。
- 封面与字幕：支持自动发现封面、手动自定义封面、字幕解析缓存、会话内字幕、播放卡片字幕和通知字幕。
- 跨页面/后台字幕显示：支持全局字幕悬浮窗，并可调整字体、字号、颜色、背景模糊、透明度和边框深度。
- 通知控制：支持单会话和多会话通知样式，提供播放/暂停、上一首、下一首、恢复通知、清空通知等控制。
- 睡眠计时器：支持手动倒计时、播放触发倒计时、到点暂停，以及暂停后在指定本地时间自动恢复播放。
- 视频转音频：基于 FFmpeg 将视频转换为 `mp3`、`aac`、`ogg`、`wav`、`flac`，支持码率选择、实时进度和取消转换。
- 应用内更新：可检查 GitHub Releases、下载新版 APK，并在需要时提前提示“允许安装未知来源应用”。
- 外观与偏好：支持深色模式、中/日/英语言切换、底部播放卡片开关、添加后自动播放、多线程播放和通知栏开关。
- 权限前置提示：涉及通知、未知来源安装、悬浮窗、后台运行、精确定时等权限的操作，会先提示再跳设置；用户返回应用后会自动继续原操作。

## 导入方式

在音频库页右下角 `+` 菜单中有三种导入方式：

| 方式 | 适合场景 | 会导入什么 | 后续刷新行为 |
|---|---|---|---|
| 导入文件夹 | 你只想监听一个固定目录 | 扫描该目录及其子目录中的受支持媒体文件，并把这个目录加入监听列表 | 下拉刷新时会重新扫描这个目录，新增根节点优先显示在顶部 |
| 导入曲库 | 你有一个“大目录”，里面每个一级子目录都是一套内容 | 会把所选根目录下的一级子目录分别加入监听；根目录里散落的媒体文件也会一并导入 | 下拉刷新时会重新扫描整套资料库结构，补齐新增文件夹/文件，并保留排除规则 |
| 导入文件 | 你只想临时或零散加入几个媒体文件 | 只导入手动选中的文件，标记为“单曲”，不绑定到某个监听文件夹 | 不参与文件夹扫描；文件仍保留在库中，直到你手动移除 |

### Android 上的实际导入策略

- 导入文件夹/导入资料库：优先使用系统文档树选择器（SAF），拿到持久化读取授权；如果设备 ROM 的文档界面不可用，会回退到旧的文件夹选择流程。
- 导入文件：优先使用系统文档文件选择器（支持多选），直接保留 `content://` 访问授权；部分设备不支持时会回退到旧文件选择流程。
- 很多机型上即使没有完整文件管理权限，也可以成功导入和播放用户明确选择的文件或目录。

## 使用说明

### 音频库

- 下拉刷新会重新扫描当前所有监听目录和资料库。
- **层级导航**：点击文件夹节点可进入下级目录，支持面包屑导航或树状回退。
- 支持搜索高亮、匹配数量提示、根目录拖拽排序、左滑移除。
- 新导入或刷新同步出的根目录会排在当前列表最上方，已有手动排序仍会保留。
- 可切换“全部 / 标签 / 声优 / 社团”分类视图，按作品详情中的字段筛选内容。
- 分类页顶部的标签、声优、社团元素支持长按复制文本。
- 资料库存在时，可进入“编辑资料库”页面，对文件夹或单曲做排除/恢复；页面会保留完整文件树，不会因为排除父级而隐藏子结构。
- 文件夹和单曲都可以打开作品详情，编辑元数据、重命名目标名称、从 DLsite 获取信息。

### 播放列表与详情页

- 从音频库点击文件或文件夹即可创建播放会话。
- 新创建的播放会话会显示在播放列表最上方。
- 播放列表支持拖拽排序、批量暂停全部、清空全部。
- 详情页支持左右滑动切换相邻会话、下拉关闭详情页。
- 详情页支持 5 秒快进/快退、缓冲进度显示、剩余时间显示、会话独立音量和声道互换。
- 循环模式包括单曲循环、当前文件夹顺序、当前文件夹随机、跨文件夹顺序、跨文件夹随机。

### 封面

- 自动封面会优先查找常见文件名，如 `cover`、`folder`、`front`、`album`、`artwork`、`poster`。
- 对视频媒体可提取视频帧作为封面候选。
- 长按详情页封面可以进入自定义封面选择模式，左右滑动浏览当前音频所在根目录中的图片并确认替换。
- 自定义封面会同步刷新详情页、音频库卡片、播放列表卡片和底部播放卡片。

### 字幕

- 支持解析常见字幕文件，如 `.srt`、`.ass`、`.ssa`、`.vtt`、`.lrc`。
- 可在详情页三点菜单里打开/关闭字幕。
- 底部播放卡片和通知可以显示当前字幕文本。
- 启用“字幕全局显示”前，会先提示开启悬浮窗权限；回到应用后会自动继续该操作。
- 字幕悬浮窗样式可在设置页中调整并持久化保存。

### 睡眠计时器

- 支持手动倒计时和播放触发倒计时两种模式。
- 到点后统一走 Android 原生执行链路暂停播放。
- 可选择在指定本地时间自动恢复被计时器暂停的会话。
- 计时器页和设置页会显示通知权限、精确闹钟、电池优化状态，帮助判断超长后台场景下的可靠性。

### 视频转音频

- 可选择源视频、输出目录、目标格式和码率。
- 支持 `mp3`、`aac`、`ogg`、`wav`、`flac` 输出；`wav` 和 `flac` 不使用码率参数。
- 转换时显示进度，可随时取消。
- 输出文件会避免覆盖已有文件。

### 应用内更新

- 可在设置页检查 GitHub 最新版本。
- 下载更新前会先检查“允许安装未知来源应用”权限。
- 若系统尚未授权，会先弹出提示框，再跳转系统设置；用户返回应用后会自动继续安装流程。

## 权限说明

| 权限 | 是否必须 | 用途 |
|---|---|---|
| `READ_MEDIA_AUDIO` / `READ_EXTERNAL_STORAGE` | 视机型而定 | 兼容直接文件系统扫描与旧式文件选择流程 |
| `MANAGE_EXTERNAL_STORAGE` | 否 | 仅用于需要完整直接文件系统访问的机型/路径；不是 SAF 导入的前提 |
| `POST_NOTIFICATIONS` | 推荐 | 播放通知、后台控制、状态提示 |
| `SYSTEM_ALERT_WINDOW` | 仅字幕全局显示需要 | 允许悬浮字幕跨页面/后台显示 |
| `FOREGROUND_SERVICE` / `FOREGROUND_SERVICE_MEDIA_PLAYBACK` | 是 | 后台与息屏播放 |
| `WAKE_LOCK` | 是 | 降低息屏后 CPU 过早休眠导致播放中断的概率 |
| `REQUEST_IGNORE_BATTERY_OPTIMIZATIONS` | 推荐 | 引导用户允许后台运行/忽略电池优化 |
| `SCHEDULE_EXACT_ALARM` / `USE_EXACT_ALARM` | 推荐 | 提升定时暂停和自动恢复在长时间后台场景下的可靠性 |
| `RECEIVE_BOOT_COMPLETED` | 推荐 | 系统重启或应用更新后恢复计时器运行状态 |
| `REQUEST_INSTALL_PACKAGES` | 仅应用内更新需要 | 下载新 APK 后触发系统安装流程 |
| `INTERNET` | 仅更新检查和 DLsite 信息获取需要 | 检查 GitHub Releases、下载更新、获取 DLsite 元数据 |

## 项目结构

```text
lib/
  i18n/                         中/日/英多语言文案
  models/                       MusicTrack、LibraryNode、LibraryEntry、PlaybackMode、作品详情模型
  providers/                    AudioProvider 门面与功能拆分
  screens/                      音频库、播放列表、计时器、设置、视频转音频、DLsite 信息确认
  services/                     SQLite、Native 桥接、更新、字幕、通知、权限控制、DLsite 元数据
  theme/                        主题与深色模式
  widgets/                      通用组件与业务组件

android/app/src/main/kotlin/    原生播放、通知、扫描、计时闹钟、保活服务、字幕悬浮窗
third_party/audio_service/      项目内维护的 audio_service fork
test/                           数据库、Provider、通知、计时器、播放桥接等测试
```

## 发行说明 v0.8.71

- **层级媒体库系统**：重构媒体库扫描与管理逻辑，引入 `LibraryEntry` 系统，支持文件夹分级浏览和细粒度的条目级状态管理。

- **息屏保活优化**：精细化 WakeLock 唤醒策略，新增原生层详细播放状态与系统资源占用日志，大幅提升超长后台播放稳定性。

- **稳定性修复**：修复了某些机型下息屏后播放队列中断及媒体库重复扫描的问题。
